# 채팅 기능 개선 및 마크다운 렌더링 계획

## 목표
채팅 경험을 개선하기 위해 대화 초기화 기능과 AI 응답의 가독성을 높이는 마크다운 렌더링을 도입합니다.

## 변경 사항

### 1. 새 채팅 (New Chat) 기능
*   **프론트엔드 (`Chat.jsx`)**:
    *   헤더에 "새 채팅" 버튼 추가.
    *   버튼 클릭 시 현재 대화 목록(`messages`) 초기화 로직 구현. (백엔드 세션이 없다면 프론트엔드 상태 초기화).

### 2. 마크다운 렌더링
*   **라이브러리**: `react-markdown`, `remark-gfm` 설치.
*   **프론트엔드 (`Chat.jsx`)**:
    *   메시지 표시 영역을 일반 텍스트에서 `<ReactMarkdown>` 컴포넌트로 교체.
    *   코드 블록, 테이블, 리스트 등의 포맷팅 지원 확인.

## 검증 계획
*   채팅 도중 "새 채팅" 버튼 클릭 시 대화 내용 사라지는지 확인.
*   AI가 생성한 코드나 목록이 마크다운으로 깔끔하게 렌더링되는지 확인.
