# 법적 고지 (Legal Notice) 페이지 구현 계획

## 목표
서비스 이용 약관, 개인정보 처리방침, 채팅 저장 정책 등을 명시하는 법적 고지 페이지를 추가합니다.

## 변경 사항

### 1. 프론트엔드
*   **컴포넌트**: `src/components/LegalNotice.jsx` 생성.
    *   내용: 면책 조항, 데이터 프라이버시, 채팅 기록 정책(SessionStorage 사용 등).
*   **라우팅**: `App.jsx`에 `legal` 뷰 상태 추가.
*   **접근성**: `MainLayout.jsx` 사이드바 혹은 헤더에 법적 고지 페이지로 이동하는 링크 추가.
*   **다국어 지원(i18n)**: 영어/한국어 번역 키 추가 (`translation.json`).

## 검증 계획
*   사이드바 메뉴에서 법적 고지 페이지로 이동 확인.
*   언어 변경 시 내용 번역 확인.
